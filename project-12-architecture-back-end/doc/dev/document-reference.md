# Développez une architecture back-end sécurisée avec Python et SQL
Projet Openclassrooms N°12. Formation "Développeur d'application Python".
Vous allez concevoir une architecture back-end sécurisée avec une base de données relationnelle, mettre en place l’authentification des utilisateurs, modéliser les données (ERD) et utiliser des outils comme Sentry pour le suivi des erreurs.

## Description du projet
Création d'un système CRM sécurisé interne à l'entreprise, qui nous aiderait à collecter et à traiter les données des clients et de leurs événements. Notre objectif principal est de mettre en place une base de données qui permette de
stocker et de manipuler de manière sécurisée les informations de nos clients, ainsi que les contrats et les événements que nous organisons. 

## Technologies utilisées
- Python
- SQLite
- Sentry
